## Automatic Installation Skelpy Node

### Create New User

```
adduser your_user

adduser your_user sudo

su - your_user 
```

### Download and Install

```
wget https://raw.githubusercontent.com/SkelpyCoin/skelpy-node/master/SkelpyCommander.sh

bash SkelpyCommander.sh
```
